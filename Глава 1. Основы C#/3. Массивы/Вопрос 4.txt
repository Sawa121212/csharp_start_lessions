Дан следующий массив

int[][] nums = new int[3][];
nums[0] = new int[2] { 1, 2 };
nums[1] = new int[3] { 3, 4, 5 }; 
nums[2] = new int[5] { 6, 7, 8, 9, 10 };
Каким образом мы можем обратиться к числу 7 в этом массиве?

Варианты ответов:
nums[7]
nums[2, 1]
nums[2][1]
nums[3]
nums[3, 3]
nums[3][3]
